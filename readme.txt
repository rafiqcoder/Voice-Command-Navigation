A plugin that enables voice commands for navigation on your WordPress site.

== Description ==
This plugin adds a button to your WordPress site that allows users to navigate using voice commands like "home", "about", and "contact".

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/voice-command-plugin` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
